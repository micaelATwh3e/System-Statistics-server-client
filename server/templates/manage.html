Setup Instructions

  1. Install client on target computer: Copy the client.py file to the computer you want to monitor.
  2. Install dependencies: Run pip install fastapi uvicorn psutil requests on the target computer.
  3. Start the client: Run uvicorn client:app --host 0.0.0.0 --port 8000 on the target computer.
  4. Get the token: The client will print a token when it starts. Copy this token.
  5. Add to dashboard: Use the form above to add the computer with its name, URL, and token.

Note: Make sure the client computer is accessible from this server and that any firewalls allow the connection.
